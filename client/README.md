# WorkZen HRMS — Client

React 19 + TypeScript + Vite 8 + Tailwind CSS 4

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Design System

Neobrutalism style — see `src/index.css` for custom classes:
- `.neo-card` — Card with thick border and offset shadow
- `.btn-neo` — Mulberry button with shadow
- `.btn-neo-secondary` — White button with border
- `.input-neo` — Input with thick border

See [root README](../README.md) for full documentation.
